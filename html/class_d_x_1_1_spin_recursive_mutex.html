void DX::SpinRecursiveMutex::lock ( ) const
virtual

Locks the mutex. Further calls to lock will block until there is a call to unlock().

Note
lock() is not recursive.
lock() is blocking.

bool DX::SpinRecursiveMutex::tryLock ( ) const
virtual

Attempts to lock the mutex. tryLock() is non-blocking, meaning that regardless of the state of the mutex when tryLock() is called, execution will continue.

Note
tryLock() is not recursive.
tryLock() is non-blocking.
